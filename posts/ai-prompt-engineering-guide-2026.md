## プロンプトとは？

**プロンプト**とは、AIへの「指示文」のことです。同じAIを使っても、プロンプトの書き方次第で回答の質が大きく変わります。

「うまく使えない」と感じている人の多くは、AIが悪いのではなく**プロンプトの書き方**に改善の余地があります。このガイドでは、今日から使えるプロンプトの基本と実例を紹介します。

---

## 良いプロンプトの3原則

### 1. 役割を与える（ペルソナ設定）

AIに「誰として答えるか」を指定すると、回答の専門性が上がります。

**Before（悪い例）：**
```
マーケティングについて教えて
```

**After（良い例）：**
```
あなたは10年以上のBtoB SaaS企業のマーケティング経験を持つ専門家です。
中小企業が限られた予算でリード獲得を増やす方法を教えてください。
```

---

### 2. 出力形式を指定する

回答の形式（箇条書き・表・文章など）を指定すると、そのまま使える回答が得られます。

**例：**
```
以下の内容を、初心者向けに：
・1段落の概要
・5つの箇条書きの要点
・締めの一言

という形式でまとめてください。
```

---

### 3. 背景・文脈を伝える

AIはあなたの状況を知らないので、必要な背景情報を提供しましょう。

**例：**
```
私は中小企業（社員20名）の総務担当者です。
来月から在宅勤務制度を導入することになりました。
社内向けの利用規則（簡易版）の草案を作成してください。
法律的な正確さより、社員が読みやすいことを重視してください。
```

## シーン別プロンプトテンプレート

### 文章作成・ライティング

```
以下の条件でブログ記事の導入文を書いてください：
- テーマ：[テーマ]
- ターゲット読者：[読者層]
- 文字数：300〜400文字
- トーン：[親しみやすい / 専門的 / カジュアル]
- 必ず含めたいキーワード：[キーワード]
```

### 情報の要約・整理

```
以下の文章を読んで、
1. 3行で要約
2. 重要なポイントを5つ箇条書き
3. 疑問に思った点や不明な点

の3つに分けて整理してください。

[文章をここに貼り付け]
```

### アイデア出し・ブレインストーミング

```
[テーマ/課題] について、
一般的なアイデアではなく「意外性がある」または「競合がやっていない」
視点からアイデアを10個出してください。
実現可能性より独自性を重視してください。
```

### コードの作成・デバッグ

```
以下のPythonコードにバグがあります。
エラーメッセージも貼り付けますので、
原因と修正方法を教えてください。

【コード】
[コードをここに貼り付け]

【エラーメッセージ】
[エラーをここに貼り付け]
```

## よくあるNG例と改善方法

| NG例 | 問題点 | 改善例 |
|------|--------|--------|
| 「副業のアイデアを教えて」 | 漠然としすぎ | 「在宅・初期費用1万円以内・週10時間以内でできる副業を5つ」 |
| 「メールを書いて」 | 情報不足 | 「取引先への納期遅延のお詫びメール（担当者向け・丁寧なトーン）」 |
| 「改善して」 | 何を改善？ | 「この文章を、説得力を高めながら半分の長さに要約して」 |

## 上級テクニック：Chain-of-Thought（思考の連鎖）

複雑な問題には「**段階的に考えてください**」と加えると精度が上がります。

```
以下の問題を、ステップバイステップで考えながら解いてください。
途中の思考過程も示してください。

[問題]
```

---

## まとめ

良いプロンプトの基本：

1. **役割を与える** → 「あなたは〇〇の専門家です」
2. **出力形式を指定する** → 「箇条書きで」「表で」「300字で」
3. **背景・文脈を伝える** → 「私は〇〇で、〇〇という状況です」

最初は難しく感じるかもしれませんが、これらを意識するだけでAIの回答質は劇的に変わります。まずは今日のメール1通から試してみてください。
